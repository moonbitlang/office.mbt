# PDFlite Fixture Notes

This directory contains a small, checked-in subset of fixture artifacts copied
from the cpdf source corpus. These files are used by acceptance tests so a fresh
checkout does not depend on the local reference checkout under `.repos`.

The copied upstream files retain their original licensing. See `LICENSE.md` for
the upstream license text. The upstream `README.md` is kept at its original path
because some tests use it as a fixture payload.
